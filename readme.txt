Nettoyage et optimisation automatique et manuelle de la base de données.

== Description ==
= French =
* Le nettoyage des tables wp_post et wp_postmeta ainsi que l"optimisation de toutes les tables de données de votre site sont réalisées chaque jour automatiquement après minuit.
* Le nettoyage peut être effectué manuellement à partir du tableau de bord.	
* Ce plugin est très léger, il est chargé en mémoire PHP uniquement lors de la visite du tableau de bord et donc ne ralentit pas le chargement des pages de votre site web.
* Il est écrit en 62 lignes de code PHP et est extrêmement léger (environ 0,03 Mo).
* Ce plugin est un dérivé de l'excellent Clean Up https://wordpress.org/plugins/clean-up/

= English =
* Cleaning and wp_postmeta tables wp_post well as "optimization of all data tables are made from your site automatically every day after midnight.
* Cleaning can be done manually from the dashboard.
* This plugin is very light, it is loaded into PHP/'memory only when visiting the dashboard and therefore does not slow down the loading of pages on your website.
* It is written in 62 lines of PHP code and is extremely lightweight (about 0.03 MB).
* This plugin is a derivative of the excellent Clean Up https://wordpress.org/plugins/clean-up/

== Installation ==

1. Dézippez l'archive et placez là dans le dossier /wp-content/plugins
1. Activez le 'Plugin' depuis le tableau de bord de WordPress
